# 🎼 Музыкальное приложение

Интерактивный тренажёр для учеников музыкальной школы. Работает прямо в браузере — без сервера, без установки.

**[Открыть приложение →](https://your-username.github.io/your-repo-name/)**

---

## Модули

| Модуль | Описание |
|--------|----------|
| 🎵 Нотная грамота | Ноты на стане, ритм, длительности, построение интервалов |
| 🥁 Ритмический тренажёр | Группировка длительностей, правила бимования |
| 🎹 Цифровка | Цифровое обозначение аккордов, бас-мелодия |
| 👂 Слуховой анализ | Интервалы, аккорды, цепочки на слух |
| 🎼 Музыкальные диктанты | Мелодические и двухголосные диктанты |
| 🔑 Тональности | Квинтовый круг, ключевые знаки, викторина |
| 📚 Муз. литература | Композиторы, произведения, жанры |
| 📖 Словарик терминов | Карточки, квиз, мэтчинг терминов |
| 💬 Цитатник | Цитаты великих музыкантов |

## Возможности

- 🌗 Светлая и тёмная тема
- 🌐 Русский и английский язык
- 📱 Адаптивный дизайн (телефон, планшет, компьютер)
- 💾 Настройки сохраняются между сессиями

## Запуск

**Локально** — скачайте репозиторий и откройте `index.html` в браузере.

**GitHub Pages** — форкните репозиторий, включите Pages в настройках (Settings → Pages → Branch: main), и приложение будет доступно по адресу `https://your-username.github.io/your-repo-name/`.
